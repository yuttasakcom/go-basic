# Go Basic

## Go Init
```
go mod init gitlab.com/yuttasakcom/go-basic
```

## Git Init
```
git init
```

## Git Add Origin
```
git remote add origin git@github.com:yuttasakcom/go-basic.git
```

## Git View Remote
```
git remote -v
```

## Git Add
```
git add .
```

## Git Push
```
git push --set-upstream origin main
```
